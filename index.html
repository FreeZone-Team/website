Features of VIPER
We provide the best features for your server!
Welcomer

VIPER has a beatiful welcome plugin. All messages and log channels can be customized from dashboard. This plugin also have join role function.

Moderation

VIPER has a lot of moderation commands such as ban, kick, mute and other. We try to make your server safe as possible with all our commands and auto-mod.

Music

VIPER can even play music. It has commands such as play, stop, pause, resume, queue, etc. You can use a link or search a song by its name. The quality is great.

Dashboard

VIPER has interactive dashboard with which you can customize messages, toggle plugins, select log channels, etc. The whole dashboard works with AdminLTE.

Support

Almost every day we add new features, fix bugs, respond to users, so the possibilities are endless. If you have an idea for our bot you can easily send it via our support server.
